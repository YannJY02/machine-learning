# Resources

Use this registry to seed study units from official course materials and widely verified open-source references. Do not copy large upstream content into this repo; link to it, cite it in notes, and write local understanding in your own words.

Checked: 2026-06-29

## Source Priority

1. Official Hung-yi Lee course pages and official/sample-code repositories.
2. High-reputation open-source tutorials or aggregations with strong community signals.
3. Personal solution repositories only for comparison after attempting the exercise.

## Course Track Strategy

- **Core track**: Machine Learning 2021 Spring.
- **Extension track**: Machine Learning 2023 Spring for GenAI-related updates after matching core topics are covered.
- **Deferred**: 2025/2026 Spring AI Agent and LLM-specialized material until the core track is stable.

## Local Mirrors

- [2021 Spring resources](./2021-spring/README.md): structured manifest and ignored local downloads for the core track.

## Exclusions

- Do not use low-signal personal homework-solution repositories as the first source.
- Do not copy assignment answers before making an independent attempt.
- Do not commit mirrored datasets, checkpoints, or bulk generated outputs from upstream sources.

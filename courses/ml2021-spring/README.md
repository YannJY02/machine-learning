# Machine Learning 2021 Spring

Core track: Hung-yi Lee Machine Learning 2021 Spring.

Source of truth: <https://speech.ee.ntu.edu.tw/~hylee/ml/2021-spring.php>

## Study Loop

1. Check `progress.md` for the current unit.
2. Read the official PDF from `resources/2021-spring/`.
3. Write understanding in the unit `notes.md`.
4. Do the homework in `practice/`.
5. Record mistakes, open questions, and useful results in `review.md`.

## Unit Layout

```text
units/001-topic-name/
|-- notes.md
|-- assignment.md
|-- practice/
`-- review.md
```

- `notes.md`: concept notes in your own words. Chinese and English may be mixed; technical terms stay in English.
- `assignment.md`: homework goal, source files, local assets, and submission/review checklist.
- `practice/`: runnable notebooks or scripts. Start with notebooks; add scripts only when repeated execution needs them.
- `review.md`: mistakes, open questions, experiment records, and next actions.

Do not commit mirrored datasets, checkpoints, generated output batches, or long logs.

# Discussions

Use this folder for group discussion records. Keep discussion notes separate from lecture notes, then move stable conclusions back into the relevant study unit.

Suggested file name:

```text
YYYY-MM-DD-topic.md
```

Suggested structure:

```markdown
# Discussion: Topic

## Participants

## Questions

## Explanations

## Follow-ups
```

- Put unresolved questions into the related study unit's `review.md`.
- Move stable explanations into the related study unit's `notes.md`.
- Keep technical terms in English.
